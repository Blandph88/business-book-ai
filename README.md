# Business Book

Turn your LinkedIn connections export into a working BD pipeline — **private, on your machine.**

Business Book is a local-first BD CRM for independent consultants and advisors. Import your
LinkedIn `Connections.csv` (and, optionally, your messages export), and it classifies every
contact by **industry, function and seniority**, then gives you the pipeline tooling to work
them: dashboard, metrics, contacts, meetings, opportunities, and revenue/SoW tracking.

Your network lives in a file **you own** — never in a cloud. Distributed via
[Freehold](https://freeholdbase.com).

## How it works

There's no server and no data collection. Everything — import, classification, and your
edits — happens in the browser and is persisted to local storage you control.

- **In-browser classifier** (`src/data/classify.ts`) — `classifyContact({company, title})`
  derives four independent dimensions from PII-safe fields (company name + job title only):
  - **sector** — company → industry via an exact dictionary, then whole-token fuzzy matching,
    then keyword/regex rules, falling back to `Other / Smaller firms` for the long tail.
  - **function** — ~25 ordered, cross-industry functions, each compiled to a whole-word regex
    (first match wins), with `Other Functions` always sorted last.
  - **seniority** — five rule-based bands (see below), derived from the title independently of
    function (a contact in `Other Functions` still gets a seniority).
- **The dictionary is the accuracy ceiling.** Because nothing phones home, classification
  quality is set entirely by the dictionaries and rules in `src/data/companies/` and
  `src/data/classify.ts` — there's no usage flywheel to lean on.

### The fixed axes (`src/data/vocab.ts`)

- **Industries** (`SECTOR_GROUPS`): Financial Services · Technology · Healthcare & Pharma ·
  Energy & Industrial · Consumer & Retail · Public Sector · Professional Services · Real Estate,
  plus `Independent / Self-employed` and the `Other / Smaller firms` catch-all.
- **Seniority** (`SENIORITY`): Executive Leadership · Head of / Director · VP / SM · Manager ·
  Associate / Analyst.
- **Service lines** (`SERVICE_LINE`): Strategy · Operations · Technology · Risk & Compliance ·
  Finance & Deals · People & Change · Data & Analytics · Other.

## Modes

- **Demo** — a zero-PII sample network, generated by running plausible synthetic contacts
  through the *same* classifier. Regenerate with `npm run gen-demo` (writes `public/*.csv`).
- **Owned** (`?mode=owned`) — the buyer imports their real `Connections.csv` via the in-app
  import flow; rows are classified and persisted to local storage (IndexedDB). Re-importing a
  newer export re-runs the classifier. Stale labels from earlier classifier versions are healed
  on load (`src/data/contacts.ts`).

## Run

```
npm install
npm run gen-demo   # writes public/*.csv (zero PII)
npm run dev        # http://localhost:5199  (or: npm run build)
npx tsc --noEmit   # typecheck
```

Desktop-first: the full data-owning experience needs a laptop; on mobile the app is
read-optimised (see `MobileNote`).

## Layout

- `src/App.tsx` — the top app-bar (brand · nav · currency) + the active tab panel.
- `src/tabs/` — Dashboard, Metrics, Contacts, Meetings, Opportunities, Revenue & SoW.
- `src/data/` — classifier, vocab, company dictionaries, CSV loading.
- `src/components/` — shared UI (brand/logo, import modal, pipeline matrices, account view…).
